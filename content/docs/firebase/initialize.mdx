## Obtain your Firebase config object
Follow this [link](https://firebase.google.com/docs/web/learn-more?_gl=1*13ueiyd*_up*MQ..*_ga*MTYwNDA0MDQ4OC4xNzYyMDQ4MjMz*_ga_CW55HF8NVT*czE3NjIwNDgyMzMkbzEkZzAkdDE3NjIwNDgyMzMkajYwJGwwJGgw#config-object) to learn about firebase config object and obtaining them.

<Callout>
	We recommend storing your app's Firebase project configuration in environment variables. TernSecure instances will read these env vars and handle Firebase initialization for you.
