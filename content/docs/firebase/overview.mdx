---
title: Firebase Guides
description: "How to configure Firebase so TernSecure Authentication works correctly. Because TernSecure is built on top of Firebase Authentication, misconfigured Firebase settings will prevent authentication from working. Complete the console steps: provide correct SDK configuration so Firebase can initialize, enable the required sign-in providers, add authorized domains, and set the required redirect URIs for redirect flows."
---

## What you'll need

- A Firebase project with a registered Web app
- App's Firebase project configuration
- At least one provider enabled (for testing, Email/Password or Google)
